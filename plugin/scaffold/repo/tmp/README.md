# tmp（作業領域・Git管理外）

下書き・作業メモなど、**リポジトリに残さないファイル**を置くディレクトリです。この README 以外の中身は `.gitignore` によりコミットされません。

## 用途

- 定例前のアジェンダ案（`prepare-meeting` 等スキルの出力先）
- 作業メモ・TODO・技術調査メモ
- 一時的なデータ加工・スクリプトの出力
- 他の場所へ移す前の下書き

## 注意事項

- ここに置いたファイルは**正式な成果物ではありません**（コミットされないため、他のメンバーからは見えません）
- 残すべきものは適切なディレクトリ（`課題管理/`・`共有資料/`・`Cortex/` 等）へ移動してください
- 課題管理ツール（Backlog 等）に登録するものは、ここで下書きしてツール側へ登録します。登録後はツール側が正本になり、同期でリポジトリに取り込まれます
