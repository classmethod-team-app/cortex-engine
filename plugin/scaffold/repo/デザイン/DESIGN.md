# DESIGN.md — {{プロジェクト名}}

## Overview

Cortex 既定の「ニュートラル / Liquid Glass」テーマ。近未来的で軽快、読みやすさを最優先にする。白を基調に、薄い青のグラデーション背景（#FFFFFF → #F4F8FE → #DCEAFD）を敷き、その上に半透明のすりガラスパネルを重ねて奥行きを出す。ブルー #2563EB を唯一の主役アクセントとして主要アクション・見出しに使い、黄・ピンク・緑は装飾（状態バッジ等）に限定する。密度はゆったり、声色は落ち着いて明快。

## Colors

パレットの意図を役割ごとに示す。純黒（#000）・純白の面での多用は避け、青みを含む近似色で軽さを保つ。

- **Primary (#2563EB):** 主要アクション・見出し・リンクに使う。画面の視線誘導の主役。
- **Primary strong (#1D4ED8):** primary の押下・ホバー時の濃い状態。グラデーションの終点にも使う。
- **Surface (#FFFFFF):** パネル・カードの面。実際には半透明（0.55〜0.72）にしてガラス化する。
- **Background (#F4F8FE):** 画面全体の薄青背景。上部を白に寄せたグラデーションで空気感を出す。
- **Text (#1B1F24):** 本文・見出しの近似黒。純黒 #000 は硬いので避ける。
- **Text muted (#6B727C):** 補足・キャプション・非活性ラベル。
- **Border (#DBEAFE):** 面の境界・区切り。薄い青で主張しすぎない。
- **Accent yellow / pink / green (#FACC15 / #F472B6 / #4ADE80):** 状態バッジ等の装飾のみ。主役の青と競わせない。

## Typography

Noto Sans JP を主体にする。見出しは太め（700）で明快な階層を作り、本文は 15px・行間 1.7 で読みやすさを確保する。ラベル・キャプションは text-muted で本文と差をつける。1画面でウェイトを増やしすぎない（本文400・強調500〜600・見出し700 の範囲に収める）。

## Layout

単一カラムを基本にし、余白を広めに取る。最大幅は 960px を目安にして中央寄せする。パネル内側の余白は 28px、要素間は 16〜24px を基準スペーシングにする。詰め込まず、空白で情報の区切りを作る。

## Elevation & Depth

影は濃く使わず、深度は Liquid Glass（すりガラス）で表現する。パネルは背景を透かしてぼかす：`backdrop-filter: blur(20px) saturate(1.5)`。上端に内側ハイライト `inset 0 1px 0 rgba(255,255,255,0.8)` を入れ、ガラスのエッジを光らせる。落ち影はソフトに `0 8px 32px rgba(16,24,40,0.10)` まで。背景のグラデーションがパネル越しに透けて見える状態を保つ。

## Shapes

角丸は3段に統一する：パネル/カード 20px（rounded.lg）、ボタン/入力 12px（rounded.md）、バッジ 9999px（rounded.full）。鋭角は使わず、全体をやわらかい印象でそろえる。

## Components

- **Button (Primary):** #2563EB → #1D4ED8 のグラデーション塗り、白文字、12px 角丸。内側に上端ハイライトを入れてガラスの艶を出す。1画面につき最も重要な1アクションだけに使う。
- **Button (Ghost):** 半透明のガラス面（塗りは薄く）に #1D4ED8 の文字。副次アクション用。
- **Card / Panel:** 半透明の白（0.55〜0.72）＋ backdrop-blur、上端の白ハイライト、20px 角丸。背景の薄青グラデを透かす。
- **Badge:** 半透明すりガラスのピル（9999px 角丸）。黄=進行中／ピンク=要確認／緑=完了 の状態表現に使う。

## Do's and Don'ts

- Do ブルー #2563EB は主要アクション・見出しに限定して使う。
- Don't 黄・ピンク・緑を主役にしない（装飾・状態表現のみ）。
- Do 本文は近似黒 #1B1F24 を使い、純黒 #000 は避ける。
- Don't 影を濃くしてガラスの透明感を損なわない。
- Do 角丸は 12px / 20px / 9999px の3段にそろえる。
