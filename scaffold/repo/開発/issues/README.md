# 開発Issue

GitHub Issues は**このリポジトリにミラーしません**。正本は常に GitHub 側です（古いコピーを作らないための方針）。

- 一覧: https://github.com/{{開発リポ}}/issues
- Claude Code から取得:
  - 一覧: `gh issue list --repo {{開発リポ}}`
  - 個別: `gh issue view <番号> --repo {{開発リポ}}`

> GitHub は `gh` CLI（や GitHub MCP）でライブ参照できるため、Backlog 等と違ってエクスポートを置きません。最新の状態が必要なときは上記で直接取得してください。
